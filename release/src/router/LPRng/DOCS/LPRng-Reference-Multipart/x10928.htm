21.5. LPD Messages

These are used to indicate LPD startup or change in operation.

    Decode: lpd=host=h4.private%0aprocess=1672%0aupdate_time=1999-03-23-20:5
    1:10.507%0avalue=Starting%0a
    host=h4.private
    process=1672
    update_time=1999-03-23-20:51:10.507
    value=Starting
    lpd: 'Starting'
